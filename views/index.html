Image Search Abstraction

User Stories:

  1. I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  2. I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  3. I can get a list of the most recently submitted search strings.

To make a search:

https://boiling-beyond-48849.herokuapp.com/search/jumanji
https://boiling-beyond-48849.herokuapp.com/search/jumanji?offset=2

To see recent searches:

https://boiling-beyond-48849.herokuapp.com/recentSearches
